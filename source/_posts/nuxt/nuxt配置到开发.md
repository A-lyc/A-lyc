##  一、为什么要用Nuxt.js
原因其实不用多说，就是利用Nuxt.js的服务端渲染能力来解决Vue项目的SEO问题。
<!-- more -->
####    1：安装
npx create-nuxt-app <project-name>
cd <project-name>
进入环境命令：
npm run dev
打包编译
npm run dev
之后生成 dist（vue）和.nuxt（服务器，四个文件：.nuxt,state,nuxt.confige.js,package.json）
上传服务器运行 
npm i 之后 npm start - 项目可以跑起来了
