## 步骤：

- 1、win10 x64系统，hosts文件位置：

- C:\Windows\System32\drivers\etc

- 修改hosts这个文件 ，使用管理员打开，

- 最后一行添加 127.0.0.1 csgo.skins.com
意思就是127.0.0.1  这个指向csgo.skins.com这个本地域名，打开的时候需要端口号

## node修改vue.config.js
```
 devServer: {
        disableHostCheck: true,// 必须添加，要不然报错，无效的头部好像是
        proxy: {
        }
    }
```
